 # ✦ LUMIÈRE — Luxury Salon & Experience Website

> A premium, modern, and fully responsive front-end website designed to deliver an elegant digital experience for high-end salons and beauty brands.

---

## ✨ Overview

**LUMIÈRE** is a luxury-inspired front-end project crafted with attention to detail, smooth animations, and modern UI/UX principles.  
This project reflects a high-end salon brand identity with a focus on aesthetics, performance, and user engagement.

It is designed to simulate a real-world premium business website with interactive features like booking systems, modals, gallery filters, and smooth scroll animations.

---

## 🚀 Features

- 🎯 Fully Responsive Design (Mobile, Tablet, Desktop)
- 🎨 Premium UI/UX with Elegant Color Palette
- ⚡ Smooth Scroll & Scroll Reveal Animations
- 📅 Booking Modal with Form Validation
- 📱 Mobile Navigation (Hamburger Menu)
- 🖼️ Interactive Gallery Section
- 🧑‍🤝‍🧑 Team Showcase Section
- 💬 Testimonials Section
- 💎 Membership & Pricing Plans
- 🔔 Toast Notifications System
- ⬆️ Back to Top Button
- 🔒 Accessibility Enhancements (ARIA, Focus Handling)

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3 (Custom Properties, Flexbox, Grid)**
- **JavaScript (Vanilla JS)**

## ⚙️ Key Functionalities

### 📌 Navigation & Scroll
- Dynamic navbar styling on scroll
- Smooth anchor scrolling with offset
- Back-to-top button visibility control

### 📌 Modal System
- Booking modal with:
  - Input validation
  - Auto-focus
  - Success state UI

### 📌 Animations
- Intersection Observer for scroll reveal
- Smooth transitions using CSS variables

### 📌 User Interaction
- Toast notifications
- Membership selection toggle
- Mobile menu control
